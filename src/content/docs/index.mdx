## 最初に知っておくこと

### 立ち絵アイテムはなるべく1つにする

VTuberKitForYMM4 の **立ち絵アイテムは、それ1つがモデルを読み込んで描画しています**。

同じキャラクターで表情やセリフを切り替える場合、立ち絵アイテムを何個も増やすより、次のように使うことをおすすめします。

1. キャラクターごとに **立ち絵アイテム** を1つ配置する
2. その立ち絵アイテムに対して **表情アイテム** や **ボイスアイテム** を追加する
3. 表情、モーション、口パクなどは表情アイテム・ボイスアイテム側で指定する

この使い方のほうが、モデルの読み込み数を抑えやすく、メモリ使用量も増えにくくなります。

### 拡大率はプラグイン側で調整する

モデルの大きさを変える場合は、**YMM4側の拡大率ではなく、VTuberKitForYMM4 の立ち絵アイテム内にある拡大率**を使用してください。

YMM4側の拡大率で大きくすると、プラグインが描画した画像そのものが引き延ばされるため、表示がぼやける場合があります。

![立ち絵アイテムの拡大率を設定](/images/step-12-set-plugin-scale.webp)

初回のみ、`.ymme` ファイルを YMM4 の拡張機能として扱えるように登録します。

![ymme拡張子をWindowsに登録](/images/step-01-register-ymme-extension.webp)

### 2. `.ymme` インストーラーをダブルクリックする

BOOTH からダウンロードした `.ymme` ファイルをダブルクリックします。

![ymmeインストーラーをダブルクリック](/images/step-02-double-click-ymme-installer.webp)

### 3. YMM4にインストールする

インストール画面が開いたら、内容を確認してインストールしてください。

![ymmeインストール画面](/images/step-03-install-ymme.webp)

## 注意点

### プレビューと動画出力で見た目が変わる場合があります

プレビュー画面では操作性を優先するため、物理演算が簡易的に表示される場合があります。

動画出力時には出力用の物理演算が行われるため、プレビュー画面と動画出力結果で、髪・服・アクセサリなどの揺れものの見た目が変わることがあります。

サムネイル出力でも、プレビュー画面や動画出力時と物理演算の状態が完全には一致しない場合があります。

### 動画ファイルサイズが大きくなる場合があります

VRM SpringBone や MMD物理演算など、フレームごとに見た目が変化する要素を含む場合、通常のゆっくり立ち絵のみを使った動画に比べて、動画ファイルサイズが大きくなることがあります。

### プロジェクトファイルサイズが大きくなる場合があります

Live2D / VRM / MMD モデルは設定できる項目が多いため、使用するモデル数や設定内容によっては、YMM4のプロジェクトファイルサイズが大きくなることがあります。

### VRM 0.x モデルについて

VRM 0.x 系のモデルは、内部で VRM 1.0 形式として扱います。

一部のモデルでは、変換後に手指や肘の曲がり方が不自然になる場合があります。問題が出る場合は、Blender の VRM Add-on などで、あらかじめ VRM 1.0 形式に変換したモデルをご利用ください。

### モデルや素材の利用規約を確認してください

本プラグインにモデルデータは付属しません。

使用するモデル、モーション、音声、動画素材などは、それぞれの作者・配布元が定める利用規約に従ってください。
